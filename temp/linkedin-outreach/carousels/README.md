# Carousel Design Files

Store your Canva/Figma carousel exports here.

## Naming Convention

```
week-X-topic-name.pdf
```

Example:
- `week-1-monitoring-vs-observability.pdf`
- `week-2-three-pillars.pdf`
- `week-3-opentelemetry.pdf`

## Design Specifications

| Setting | Value |
|---------|-------|
| **Dimensions** | 1080 x 1350 px (portrait) |
| **Format** | PDF (for LinkedIn upload) |
| **Slides** | 8-10 per carousel |
| **Font Size** | Min 32pt for headers, 24pt for body |

## Canva Quick Start

1. Go to [canva.com](https://canva.com)
2. Create design → Custom size → 1080 x 1350 px
3. Use content from corresponding `week-X/XX-thu-carousel-*.md` file
4. Export as PDF
5. Save here with naming convention above

## Visual Consistency Tips

- Use same brand colors across all carousels
- Include `blog.shivam.info` watermark on final slide
- Use consistent font family (recommend: Inter, Montserrat, or Poppins)
- Keep slide transitions simple (no animations in PDF)
