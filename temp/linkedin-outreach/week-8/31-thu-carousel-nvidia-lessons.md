START SIMPLE

We didn't build the 5-layer

than we expected.

Complexity comes later.
Start working now.

### Slide 3
```
LESSON 2:
STANDARDIZE EVERYTHING

Same labels.
Same metric names.
Same dashboards.

This enables powerful
aggregations across
5,000+ nodes.
```

### Slide 4
```
LESSON 3:
INSTRUMENT FROM DAY ONE

Adding observability to
existing systems is painful.

Retrofitting context
you didn't capture?
Impossible.

Day one. Not "later."

### Slide 5
```
LESSON 4:
FEWER, BETTER ALERTS

We deleted 60% of our alerts.

Result?
Incidents got resolved FASTER.

Noise is worse than silence.

One SDK everywhere.
Switch backends without code changes.

We bet on OTel early.
No regrets.
