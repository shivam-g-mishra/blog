## How to Use Each File

Each post file contains:

1. **Metadata** - Day, time, format, target blog link
2. **POST CONTENT** - Copy-paste ready content
3. **FIRST COMMENT** - The comment with blog link (post within 60 seconds)
4. **ALTERNATIVE HOOKS** - Backup hooks if original underperforms
5. **ENGAGEMENT TIPS** - How to maximize engagement

## Posting Schedule

| Day | Time (PT) | Post Type |
|-----|-----------|-----------|
| Tuesday | 7:00 AM | Educational text post |
| Wednesday | 7:00 AM | Practical/how-to post |
| Thursday | 7:00 AM | Carousel (visual) |

## Creating Carousels

For Thursday posts, create carousel PDFs in Canva:
- **Size:** 1080x1350px (portrait)
- **Font:** Bold, minimum 32pt for headers
- **Slides:** 8-10 per carousel
- **Export:** PDF for LinkedIn upload

Save your Canva designs in the `carousels/` folder.

## Success Metrics

Track weekly:
- Impressions
- Engagement rate (reactions + comments + shares / impressions)
- Profile views
- Blog clicks (check analytics with UTM filter)
- New followers
